## Overview

TollGate-rs is the Rust implementation of the TollGate protocol for autonomous device-to-device payment for metered resource delivery. Built on Cashu ecash and Spilman payment channels.

The upstream repo is in the design phase. The Amperstrand fork contains the active implementation with M1-M3 milestones complete.
