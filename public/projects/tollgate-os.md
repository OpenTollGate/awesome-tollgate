## Overview

TollGate OS is a custom OpenWrt-based operating system designed specifically for running TollGate nodes. It turns compatible routers into Bitcoin-powered internet sharing hotspots.

## Features

- Pre-configured OpenWrt image with TollGate packages
- Captive portal for pay-per-use internet access
- Lightning Network integration for instant micropayments
- Support for multiple router models
- Automatic updates and remote management

## Supported Devices

| Device | Architecture | Status |
|--------|-------------|--------|
| GL.iNet GL-MT3000 | aarch64 | Supported |
| GL.iNet GL-MT6000 | aarch64 | Supported |

## Getting Started

1. Download the firmware for your device
2. Flash via the router's web interface
3. Configure your Lightning wallet
4. Share your internet and earn sats
