## Overview

Archipelago is a Bitcoin node infrastructure project by the Sovereign Engineering / Lab 484 ecosystem, led by Shadrach. It provides a discovery layer where TollGate nodes are discoverable and accessible on the mesh. The project is part of the decentralized "Sovereign Technology Stack" and is developed using Nostr-native tooling (NIP-34).
